# Figure and Table Plan: CSR, ESG, and Ethics Analysis of a Multinational Corporation (MNC)

Use visuals only when they clarify the argument. Every figure or table should do analytical work, not just decorate the paper.

| Planned asset | Where to use it | Purpose | What data you still need |
| --- | --- | --- | --- |
| Table 1. Requirement-to-Argument Map | Near the introduction or methodology section | Show how each brief task is being answered. | Task sections, section claims, and planned evidence. |
| Table. CSR initiative-to-theory map | Part 1 - CSR Strategy Evaluation | Link company initiatives to the selected framework clearly. | CSR initiatives, chosen theory, alignment points, and critique points. |
| Figure. ESG benchmark or materiality visual | Part 2 - ESG Performance Analysis | Summarize comparison or materiality logic at a glance. | Competitor metrics, ratings, material issues, or trend data. |
| Table. ESG comparison matrix | Part 2 - ESG Performance Analysis | Compare environment, social, and governance performance clearly. | Company KPIs, competitor KPIs, and critical commentary. |
| Table. Ethical issue gap analysis | Part 3 - Ethical Issue Deep Dive | Compare current response, evidence of progress, and remaining gaps. | Current company actions, stakeholder concerns, and ethical framework criteria. |

## Caption Checklist

- Give each figure and table a descriptive title.
- Add a source line under every non-original asset.
- If you created the visual yourself, still note the underlying data source.
- Make sure the text explicitly refers to each asset and explains why it matters.
